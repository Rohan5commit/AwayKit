# Demo Assets

This directory contains demo assets for the AwayKit project.

## Files

- `scenarios.json` — Pre-built matchday scenarios for demo mode
- `venues/` — Venue images and metadata (placeholder)
- `avatars/` — Default member avatars (placeholder)

## Adding Demo Assets

To add custom demo assets:

1. Place images in the appropriate subdirectory
2. Update the scenario data in `src/lib/demo/scenarios.ts`
3. Reference assets using relative paths
